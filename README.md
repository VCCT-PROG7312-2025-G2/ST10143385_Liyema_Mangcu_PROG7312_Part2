# MunicipalForms Web Application

##  Overview
**MunicipalForms Website** is a web application built for the community using **ASP.NET Core MVC**.  
The website allows for users ot submit reports, manage local events and announcements within their municipality. 
The goal here is to improve communication, engagement, and transparency between the community and its local authorities.

---

## Features
-  **Dashboard** ‚Äî Central hub for all municipal functions  
-  **Local Events & Announcements** ‚Äî Browse and filter community events  
-  **Report Submission** ‚Äî Submit community issue reports  
-  **Recommended Events** ‚Äî Suggestions based on recent searches done by the user 
-  **Search and Filter System** ‚Äî Quickly filter events by date or category

## Setup Instructions

### Prerequisites
- Visual Studio 2022 (or later)  
- .NET 6 or newer  

### Running the Application
1. Open the project in **Visual Studio**  
2. Restore NuGet packages if prompted  
3. Build the project (`Ctrl + Shift + B`)  
4. Run the application (`F5` or ‚ñ∂ Run button)
